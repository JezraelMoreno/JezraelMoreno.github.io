Tabla de comandos de Git y Terminal

Comando Función
mkdir [nombre_carpeta] Crea un nuevo directorio (carpeta).
touch [nombre_archivo] Crea un nuevo archivo vacío.
ls Lista los archivos y carpetas en el directorio actual.
cd [nombre_carpeta] Cambia al directorio especificado.
git init Inicializa un nuevo repositorio Git en la carpeta actual.
git clone [URL] Clona un repositorio remoto en tu computadora.
git status Muestra el estado actual de los archivos en el repositorio.
git add . Añade todos los cambios al área de preparación (staging).
git commit -m "mensaje" Confirma los cambios con un mensaje descriptivo.
git push origin main Envía los cambios confirmados al repositorio remoto en la rama main.
git pull origin main Obtiene y fusiona los cambios más recientes desde el repositorio remoto.
git branch Lista las ramas disponibles en el repositorio.
git checkout -b [nombre_rama] Crea y cambia a una nueva rama.

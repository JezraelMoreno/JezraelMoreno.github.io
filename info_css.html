¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que define la presentación de un documento escrito en HTML o XML. Permite controlar el diseño, el color, la tipografía y otros aspectos visuales de las páginas web.

Selectores en CSS

Los selectores en CSS son patrones utilizados para seleccionar los elementos HTML que se desean estilizar. Existen diferentes tipos de selectores, que se utilizan según el tipo de elementos que se quieren modificar.

Tipos de Selectores:

  • Selector por tipo: Selecciona todos los elementos de un tipo o etiqueta en el documento HTML. Ejemplo: p { color: red; } selecciona todos los elementos <p> y les aplica el color rojo.
  • Selector por clase: Selecciona todos los elementos que tienen una clase específica. Se utiliza el prefijo . antes del nombre de la clase. Ejemplo: .miClase { font-size: 16px; } selecciona todos los elementos con la clase miClase.
  • Selector por identificador: Selecciona un único elemento con un identificador específico. Se utiliza el prefijo # antes del nombre del identificador. Ejemplo: #miId { background-color: blue; } selecciona el elemento con el identificador miId.

Propiedades y Valores

Las propiedades en CSS son características que se pueden cambiar para estilizar los elementos. Los valores son los atributos que se asignan a las propiedades para definir cómo deben visualizarse los elementos.

Ejemplo:

p {
    color: blue; /* color es la propiedad, blue es el valor */
    font-size: 18px; /* font-size es la propiedad, 18px es el valor */
}

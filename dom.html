Experiencia general

La verdad es que los ejercicos salieron a la primera, tanto el html como el css, el prompt que le di fue bastante sencillo, es una herramienta util.

¿Cuando usaras ChatGPT y cuando Copilot?

Pues tal vez para cosas de generación de texto y escritura pueda usar ChatGPT y para codigo usaría Copilot, pues puede usar un modelo de Claude, que lo vuelve mucho más eficiente.

Ventajas y desventajas de usar IA para programar

En cuanto a cosas sencillas puede funcionar bastante, principalmente para comprender conceptos y como funciona codigo. Para programar algo más complejo siempre suele ser más ineficiente, pues constantemente tienes que pasar más tiempo depurando el codigo que te da, contrario a cuando lo vas escribiendo tú.
